TweakPHP is a desktop application with a code editor to tweak your PHP code.

**TweakPHP is currently Beta**

## Demo

https://x.com/saeed_vz/status/1832507929653616953

## Setup Locally

Install dependencies

```
npm install
```

Copy .env.dist as .env

```
cp .env.dist .env
```

Run the app

```
npm run dev
```
